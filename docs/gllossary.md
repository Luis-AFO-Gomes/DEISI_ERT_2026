# Glossário
- **Titulo** Edição literária, sem existência física. Identifica a obra, mas não um objecto que possa ser emprestado
- **Exemplar** Cópia física de um **Titulo**
- **Livro** O mesmo que **Título**, mas com maior abrangência, podendo referir-se a obras não existentes na plataforma.
- **Sócio**: Membro o grupo de partilhar de livros
- **Sócio contribuinte**: Sócio que contribui com livros para o sistema
- **Proprietário**: o mesmo que **Sócio contribuinte**
- **Administrador**: Membro do grupo de partilhar de livros com privilégios adicionais para gerir o sistema
- **Anónimo**: Utilizador sem conta, apenas pode consultar os títulos disponíveis
- **Candidatura**: Pedido de utilizador anónimo para se tornar sócio da plataforma
- **Disponibilidade**: Estado de um título, que pode ser "disponível" ou "indisponível", dependendo se tem exemplares disponíveis para empréstimo
